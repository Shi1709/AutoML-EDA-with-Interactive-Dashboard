## 🚀 What is this?

**AutoML Studio** is a web-based platform that automates the entire early-stage data science workflow:

- 📊 Data exploration  
- 🧹 Data preprocessing  
- 🤖 Model training  
- 📈 Performance comparison  

All in a single, interactive interface.

No heavy coding. No complex setup. Just upload your dataset and go.

---

## ✨ Why this exists

Most ML tools today are either:
- Too technical for beginners  
- Too limited for real analysis  

This project bridges that gap by combining **EDA + preprocessing + ML training** into one seamless workflow.

---

## 🔥 Core Features

### 📂 Smart Data Ingestion
- Upload datasets in multiple formats (CSV, Excel, JSON)
- Automatic schema detection
- Quick dataset overview

---

### 📊 Automated EDA (Exploratory Data Analysis)
- Statistical summaries in seconds  
- Missing value detection  
- Correlation analysis  
- Interactive visualizations  

---

### 🧠 Intelligent Preprocessing
- Auto-detects problem type (classification/regression)  
- Handles null values intelligently  
- Encodes categorical features  
- Scales numerical data  
- Optional dimensionality reduction (PCA)

---

### 🤖 One-Click Model Training
- Trains multiple baseline models instantly  
- Compares performance metrics  
- Selects the best-performing model automatically  

---

### 📈 Interactive Dashboard
- Filterable data tables  
- Real-time charts and graphs  
- Model performance comparison view  

---

### 📦 Export & Reuse
- Download cleaned datasets  
- Export trained models  
- Save reports for future use
